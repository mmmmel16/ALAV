# Sistema de gestión de mantenimiento industrial
  Esta aplicación de escritorio fue desarrollada para gestionar el mantenimiento de máquinas industriales de una empresa textil., centralizando la información que anteriormente se administraba de forma manual y dispersa. Permite registrar, organizar y planificar tareas de mantenimiento, reduciendo paradas imprevistas de producción y mejorando el control operativo de la planta.

## Problemática que resuelve
  Antes de la implementación del sistema:
- Las tareas de mantenimiento se registraban en planillas en papel o archivos sueltos.
- No existía un control centralizado del estado de las máquinas.
- Los mantenimientos preventivos no se programaban de forma eficiente.
- Se producían demoras en reparaciones y paradas no planificadas.

La aplicación resuelve estos problemas mediante un registro digital centralizado y una visualización clara de tareas pendientes.

## Tipo de aplicación
- Aplicación de escritorio
- Funcionamiento local (sin conexión a internet)
- Instalación en una única computadora
- Sin autenticación ni gestión de usuarios

## Funcionalidades principales
  La aplicación permite la gestión de:
- Máquinas
    - Alta, modificación y baja de máquinas industriales.
    - Registro del estado y características de cada máquina.

- Tareas de mantenimiento
    - Definición de tareas preventivas y correctivas.
    - Asignación de periodicidad a cada tarea.
    - Asociación de tareas a máquinas específicas.

- Órdenes de trabajo
    - Generación y seguimiento de órdenes de mantenimiento.
    - Visualización de tareas pendientes según periodicidad.

- Insumos
    - Registro de insumos necesarios para cada tarea.
    - Asociación de insumos a tareas de mantenimiento.

- Técnicos
    - Registro del personal técnico.
    - Asignación de técnicos a tareas u órdenes de trabajo.

- Mantenimiento preventivo
    - El sistema calcula automáticamente las tareas pendientes en función de la periodicidad definida, facilitando la planificación y evitando olvidos o retrasos.


## Objetivo del proyecto
  Brindar una herramienta:
- Simple
- Funcional
- Accesible
- Adaptada a la realidad operativa de una planta industrial
- Que permita mejorar la organización, planificación y control del mantenimiento industrial.
